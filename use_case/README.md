# Диаграмма вариантов использования в UML
* Данная диаграмма, отражает отношения между актерами, прецедентами и является составной частью модели прецедентов, позволяющей описать систему на концептуальном уровне.
* Для данной модели были выявлены роли: заказчик, менеджер по работе с заказчиками, менеджер по работе с заказами, сотрудник отдела производства и кладовщик.
